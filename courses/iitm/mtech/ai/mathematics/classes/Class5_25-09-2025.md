## Vector Space

Vector Space (V, F) is a set of vectors, and a field of scalars F equipped with two (binary) operations:
1. Vector Addition: x, y Є V,  (x+y) Є V    
2. Scalar Multiplication: x Є V and a Є F, a*x Є V 

Vector addition and scalar multiplication must specify certain rules.

Eight such rules to be satisfied in Vector spaces:
Got x,y,z Є R^n and c1,c2,c3
- Commutativity: x + y = y + x
- Associativity: x + (y + z) = (x + y) + z
- Zero vector: There is a unique zero vector such that x + 0 = x
- Negative vector: For each x there is a unique vector -x such that x + (-x) = 0
- Identity: 1 times x equals x
- Associativity: (c1c2)X = c1(c2X)
- Distributivity: c(x + y) = cx + cy
- Distributivity: (c1 + c2)x = c1x + c2x

## Linear Independence of Vectors

Let (V, F) be a vector space and let {v1, v2, ..., vn} be a set of vectors in V (vi Є V)
- The set of vectors {v1, v2, ..., vn} are called linearly independent, iff Є scalars, ki Є F, i = 1, ..., n such that

    k1v1 + k2v2 + ... + knvn = 0 => k1 = k2 = ... = kn = 0

In other words, the linear combination of vectors results in a zero vector, iff all the scalar multipliers are zero and that is the ONLY solution

- Conversly, the set of vectors is said to be linearly dependent iff Є scalars, ki Є F, i=1, ...,n not all zero and k1v1 + k2v2 + ... + knvn = 0

## Span

Let V be a vector space and let S = {v1, v2, ..., vn} be a subset of V. Then S spans V if every vector v in V can be written as a linear combination of vectors in S i.e.

## Basis of a Vector Space

Let (V, F) be a linear vector space. A set of vectors {b1, b2, ..., bn} is called a bases of V of 

span {b1, ..., bn} = V

and any vector v Є V can be written as a linear combination of these basis vectors as:

v = k1b1+ ... + knbn where v Є V

where {b1,..., bn} are linealy independent.

- k = [k1, ..., kn] Є F is called coordinate representation, or coordinate vector of **v** with respect to basis {b1, ..., bn}
- ki are called coordinates of **v** with respect to the basis {bi}
- **Number of basis vectors is called dimension of vector space**

Can there be another set of coordinates {k1,.., kn} that represent the same vector **v** with the given basis {b1,...,bn}? **NO**

## Orthonormal Basis

- Two vectors are orthonormal if they are orthogonal and of unit length
    v1 . v2 = 0 => dot product of two orthogonal vectors is zero
    Orthonormal vectors dot product is zero and are of unit length

- Vector set V is an orthonormal bais if each pair of vectors in V are orthonormal.

    Example:
    0.6  0.8       form an orthonormal basis of R^2
    0.8  -0.6

    1 0            form an orthonormal basis of R^2
    0 1

### Advantages of Orthonormal Basis

- Convenient basis of because the coordinates of a vector can be easily calaculated
- v Є R^n and basis of R^n = {b1,..., bn}, then ki = v.bi where i = {1,...,n}
- Standard basis is the most popularly used orthonormal basis because ki = vi

## Vector Subspace

Let (V, F) be a (linear) vector space and let U be a subset of V, U ⊂ V

Let (U, F) is called a subspace of (V, F) if (U, F) is a subspace itself.

To check if U is a subspace:
- U contains zero vector, i.e., 0 Є U
- Closure under scalar multiplication i.e., if a Є U; c Є R, then c a Є U
- Closure under addition i.e., if a,b Є U, then a+b Є U

dim U <= dim V => Dimension of U should have an upper limit of dimension of V (parent set)

Examples:

    - 0 is a 0-dimensional subspace of R^2
      0
    - span 1 is 1-dimensional subspace of R^2
           1  
    - plane is a subspace in R^3


If U1 and U2 are subspace is U1 ∪ U2 a subspace.

If U1 is X-axis  
If U2 is Y-axis  
U1 ∪ U2 -> creates a set which does not satisfies U1 and U2 - so fails to satisfy subspace criteria.  
Hence, U1 ∪ U2 is not a subspace.

If U1 and U2 are subspace is U1 ∩ U2 a subspace

If U1 is X-axis  
If U2 is Y-axis  
U1 ∩ U2 is the origin. It satisfies subspace criteria.  
Hence, U1 ∩ U2 is a subspace.  


## Linear Maps

A map f: U -> V between two vector spaces U (domain) and V (co-domain) over field F is linear if for x,y Є U and c,c1,c2 Є F, following holds:
- Homogeneity: f(cx) = cf(x)
- Additivity: f(x+y) = f(x) + f(y)
- Superposition: f(c<sub>1</sub>x+c<sub>2</sub>y) = c<sub>1</sub>f(x) + c<sub>2</sub>f(y)

Examples:
- Identity transformation: f: V -> V where f(x) = x, where x Є V
- Zero transformation: Transformation f: V -> V which maps each element of V to 0
- Multiplication by scalar: Transformation f: V -> V where f(x) = cx, where x Є V, c Є R
- Inner Product: f: V -> R such that f(x) = x.z, where x Є V, where z is a fixed vector in Eudlidean space V
- Transformation f R^n -> R^m
