## Vector Space

Let V be a set of vectors and F be a field. Then V is called a vector space over a field F if the following requirements are met:

    x, y Є V,  (x+y) Є V    
        
        [x and y are vectors in Vector set V, and resultant vector after vector addition of x and y belongs to Vector set V]
    
    x Є V and a Є F, a*x Є V 
        
        [x is a vector in Vector set V and a is a scalar; any resultant vector after scalar multipliplication of x belongs to vector set V]

__In simplified words, Vector space is a closed space over vector addition and scalar multiplication of a vector__

Example: 
- Any of the 4 quadrants on graph are independently not a Vector space 
- A given line y=1 is not a Vector space
- An entire graph inclusive of all 4 quadrants is a Vector space

## Linear Combination

Vector 

    6
    -3
    4
can be expressed as a linear combination of vectors:

    1    0    0
    0    1    0
    0    0    1


as:

    6        1        0       0
    -3 = 6 * 0 + -3 * 1 + 4 * 0
    4        0        0       1
