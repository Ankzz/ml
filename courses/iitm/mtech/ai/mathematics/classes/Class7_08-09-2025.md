## Continuous and Discrete Random Variables

Discrete Random Variables: PMF - Probability Mass Function -  
Continuos Random Variables: PDF - Probability Density Function -   

- In discrete case, the PMF reflects the probability that a specific value or a countable set of values can occur.

- In continuous case, the PDF reflects the probability of occurrence within a given range or interval

- Absolute vs likelihood: A random variable is characterized by a mathematical function (PMF/PDF) that quantifies the likelihood of the outcomes in the sample space of a random experiment.

### Simple ways of describing Random Variables

- For PMFs f(x<sub>i</sub>) = P(X = x<sub>i</sub>)   
  a) 0 <= f(x<sub>i</sub>) <= 1,  
  b) ∑<sup>n</sup><sub>i=1</sub> f(x<sub>i</sub>) = 1

- For PDFs P(a<= x <=b) = ʃ<sup>b</sup><sub>a</sub> f(x) dx:  
  a) f(x)>= 0  
  b) ʃ<sup>∞</sup><sub>-∞</sub> f(x) = 1  
  c) Probaility at a given point is 0
