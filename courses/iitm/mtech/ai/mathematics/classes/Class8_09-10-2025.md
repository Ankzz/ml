## Image and Kernel of Linear Maps
- Image of a linear map f: U -> V is:  

    im(f) = {y Є V | y = f(x) where x Є U}

- im(f) is a subspace of V as it satisfies the conditions for a subspace

- Kernel of a linear map f: U -> V is:

        ker(f) = {x | f(x) = 0 where x Є U}

    In other words, all the points x which belongs to U which maps to 0 origin in V

### System of Linear Equations as a Linear Map
- A system of m linear equations and n unknowns:

    ∑ a<sub>ik</sub><sup>n</sup>x<sub>k</sub> = b<sub>i</sub> for i=1 ... m  
    where (x<sub>1</sub>,...,x<sub>n</sub>) are the n unknowns

Consider an example:

    |  1  2 |   | x1 |    | p |  
    |  1  3 | * | x2 | =  | q |  
    |  2  1 |             | r |  
        A         X         b
    Here  this equation is of form Ax = b

    x is the point on vector subspace U
    b is the point on vector subspace V
    A is the Transformation matrix which maps f: R^n -> R^m which maps vector x onto vector b

System has:
- A unique solution if b is the image of f and kernel of f contains only zero vector  
    - f contains only zero vector can be mathematically represented as Ax = 0 and x=0
- Many solutions if b is in image(f) and kernel of f is non-trivial
    - Ax = 0 leads to infinite solutions for x
- No solution if b is not in the image(f)
    - Ax = b leads to no solution for x

Example:

    Example 1:
    A = | 1 0 | X=|x1|  b = | 2 |   
        | 0 2 |   |x2|      | 4 |

    Ax = b
    X = (x1=2, x2=2)
    In this case one unique solution
    AX= 0
    X = (x1=0, x2=0)
    Hence here, kernel for A contains zero vector

    Example 2:
    A = | 1 0 | X=|x1|  b = | 4 |   
        | 0 0 |   |x2|      | 2 |

    Ax = b
    X = (x1= 4, x2=*)
    
    AX= 0
    | 1 0 | *  | x1 | = | 0 |   
    | 0 0 |    | x2 |   | 0 |
    X = (x1=0, x2=*)

    Hence here, no solution exists

    Example 3:
    A = | 1 0 | X=|x1|  b = | 4 |   
        | 0 0 |   |x2|      | 0 |

    Ax = b
    X = (x1=4, x2=*)

### Fundamental four spaces:
- Column space C(A)
- Null space N(A)
- Row space R(A)
- Left null space or kernel of transpose of a matrix - N(A<sup>T</sup>)    

#### Column Space
- C(A) is a vector space with columns of A as its basis

        A =   | 1  0  2  2 |    
            | 0  1  1  2 |

        C(A) = | 1 | | 0 | | 2 | | 2 | 
            | 0 |,| 1 |,| 1 |,| 2 |
            
#### Null Space: N(A)
Set of vectors x such that Ax=0  

#### Row Space: R(A)
- Contains all possible linear combinations of rows of A
- Alternativelu, row space comprises of all vectors r = A<sup>T</sup>y where y E R<sup>m</sup>

            | a11  a12 |
        A = | a21  a22 |
            | a31  a32 |

                | a11 |  | a12 |
        C(A) =  | a21 |, | a22 |
                | a31 |  | a32 |


        R(A) = | a11 | | a21 | | a31 | = C(A^T)
               | a12 | | a22 | | a32 |


#### Rank

Rank of a linear map f: U-> V with corresponding m x n matrix A is:
- Dimension of the image of f ,i.e., rank(f) = dim Im(f)
- Dimension of column space of A ,i.e., rank(A) = dim C(A)
- Dimension of row space of A ,i.e., rank(A) = dim R(A)

        r = rank(f) = rank(A) = dim C(A) = dim R(A) <= min (m,n)

- Rank of A signifies number of linearly independent rows or columns of A

#### Properties of a Matrix Rank

- Matrix of full rank if: rank(A) = min(m,n)
- Matrix is rank deficient if: rank(A) < min(m,n)
- rank(A) = rank(A<sup>T</sup>)
- A n x n square matrix is invertible iff its rank is n
- rank(AB)<= min(rank(A), rank(B))
