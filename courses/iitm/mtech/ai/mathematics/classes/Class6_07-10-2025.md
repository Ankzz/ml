**Question 1**:  
For the given system of equations:  
x+y+z = 2  
x+2y+3z = 4  
x+3y+5z = 6  

a) Check whether this system has a unique solution  
b) Use your answer to justify whether coefficient vectors (1,1,1), (1,2,3), (1,3,5) are linearly dependent

**Solution**: 
a)
<pre>
1 1 1    x    2  
1 2 3 *  y =  4  
1 3 5    z    6  
Of the form Ax = b => x = inverse(A) b  
</pre>
To generate inverse of A, first check determinant of A=>  
    |A| = 1 (2*5 - 3*3) - 1 (1*5 - 1*3) + 1 (1*3-1*2) = 1 - 2 + 1 = 0  

Here, as |A| = 0, hence its not invertible.  
Perform row consistency operations:

    Row 2 = Row 2 - Row 1 = (1 2 3) - (1 1 1) = (0 1 2)
    Row 3 = Row 3 - Row 1 = (1 3 5) - (1 1 1) = (0 2 4)

    Lower 2x2 matrix is:
    1 2
    2 4

Determinant as 0 indicates dependency. So, system has infinite solutions.

b) As determinant is 0, rows are dependent, the vectors are linearly dependent.

**Question 2**

Find the basis for the subspace defined by the plane  
x + 2y + 3z -t = 0  
in R^4, what is the dimension of this subspace.

**Solution**  
Lets represent the given equation as:  

    t = x + 2y + 3z

Now factor x (x=1,y=0,z=0), basis vector would be

    x (1,0,0,1)

Now factor y (y=1,x=0,z=0), basis vector would be

    y (0,1,0,2)

Now factor z (z=1,x=0,y=0), basis vector would be

    z (0,0,1,3)

Now Basis vector set are all three.  
Entire matrix can be presented as linear combination of basis vectors as:  

(x,y,z,t) = x (1,0,0,1) + y (0,1,0,2) + z (0,0,1,3)

Here now:

    c1 (1,0,0,1) + c2 (0,1,0,2) + c3 (0, 0, 1, 3) = (0,0,0,0)
    Hence, c1 = c2 = c3 = c4 = 0
    which implies that these basis vectors are linearly independent

Based on the inference, dimensionality of the hyperplane would be 3 (as basis vector set has 3 vectors)

**Question 3**  
Let V = R^4, and define the subset  
W = {(x1,x2,x3,x4) Є R^4 : x1+2x2-x3=0, x4=0}

a) Prove W is a subspace of V  
b) Find a basis and dimension for W  

**Solution**  

a) For W to be subspace: zero vector, scalar multiplication and vector addition needs to be verified.  
Scalar multiplication - let c Є F  

    c (W) => c (a1+2a2-a3) = 0, c(a4) = 0  

Vector Addition for (a1,a2,a3,a4) and (b1,b2,b3,b4)

    (a1+2a2-a3)+(b1+2b2-b3) = 0+0 = 0

Zero Vector (0,0,0,0) satisfies:

    (0+2*0-0)=0, 0=0


b) Given x4=0  
x1 + 2x2 - x3=0 => x3 = x1 + 2x2  
Get the basis vectors utilizing the info  

Factor x1(x1=1, x2=0)

    x1 (1,0,1,0)

Factor x2(x2=1, x1=0)

    x2 (0,1,2,0)

As, there are two basis vectors - hence dimension is 2

c)  With given equations- for nullity:
